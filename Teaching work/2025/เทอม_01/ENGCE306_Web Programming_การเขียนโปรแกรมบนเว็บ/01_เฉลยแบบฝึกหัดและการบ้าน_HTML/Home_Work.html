เกี่ยวกับฉัน

สวัสดีครับ ผมสมชาย ใจดี กำลังศึกษาอยู่ที่โรงเรียนพัฒนาวิทยา มีความสนใจในการเขียนโปรแกรมและการสร้างเว็บไซต์เป็นอย่างยิ่ง ผมเชื่อว่าเทคโนโลยีสามารถสร้างสรรค์สิ่งดีๆ ให้กับโลกได้ และผมมีความฝันที่จะเป็นส่วนหนึ่งของการเปลี่ยนแปลงนั้น

ในเวลาว่าง ผมชอบศึกษาหาความรู้เพิ่มเติมเกี่ยวกับเทคโนโลยีใหม่ๆ เช่น JavaScript Frameworks และการออกแบบ UX/UI นอกจากนี้ยังชอบอ่านหนังสือและเล่นกีฬาฟุตบอลกับเพื่อนๆ ครับ

  • Microsoft Office Suite
