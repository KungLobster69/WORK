การเขียนบทความสอน HTML

หนึ่งในสถานการณ์ที่จำเป็นต้องใช้ HTML Entities มากที่สุดคือการเขียนบทความสอนเกี่ยวกับโค้ด หากเราต้องการแสดงตัวอย่างแท็ก <h1> หรือ <p> บนหน้าเว็บ เราไม่สามารถพิมพ์

ลงไปตรงๆ ได้ เพราะเบราว์เซอร์จะเข้าใจผิดว่าเรากำลังจะสร้างหัวข้อใหม่

ตัวอย่าง: การแสดง Code Snippet

เราจึงต้องแปลงอักขระ < และ > ให้เป็น Entities ก่อน ดังตัวอย่างในกล่องโค้ดด้านล่างนี้ ซึ่งใช้ทั้ง Entity Name (&lt;) และ Entity Number (&#60;) ผสมกัน 

<!-- โค้ดตัวอย่างสำหรับแสดงผลบนเว็บ -->
<div class="product-card">
  <h2>หูฟังไร้สาย รุ่น Pro</h2>
  <!-- ใช้ Entity Number สำหรับแท็ก p -->
  <p>ราคาพิเศษเพียง 2,500 บาท (ปกติ 3,000 บาท)</p>
</div>

ตัวอย่าง: การใช้อักขระพิเศษในเนื้อหาทั่วไป

นอกจากการแสดงโค้ดแล้ว เรายังใช้ Entities เพื่อแสดงสัญลักษณ์พิเศษต่างๆ ในเนื้อหา เช่น สัญลักษณ์ลิขสิทธิ์ (©), เครื่องหมายการค้า (™ หรือ ®), หรือแม้กระทั่งการบอกขนาดที่ "น้อยกว่าหรือเท่ากับ" (≤)
